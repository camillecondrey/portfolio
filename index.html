Projects

Find your dream dog

A responsive application designed to match users compatibility with different dog breeds based on quiz results. Users can search for adoptable dogs by their breed from local shelters or a location of their choice.

HTML / CSS / Javascript / jQuery / Bootstrap / AJAX

List It

List It is an application built with shopaholics like myself in mind. Users can use List It to organize the things they plan on buying into different lists. There is an opportunity to include a link for every item you add to a list and List It will scrape the link for valuable meta data like a product image, price and stock availability.

HTML / CSS / Javascript / jQuery / Node.js / Express / MongoDB / Mongoose

Mood Boost

Mood Boost offers a collection of strategies on how to get into or out of certain moods. Inspired by a gif I saw that is perfect for helping people with anxiety refocus their breathe, I thought it would be great to have useful tools like this all in one place for users who find themselves in negative moods on a regular basis.

React / Redux / Javascript / CSS / Node.js / Express / MongoDB
